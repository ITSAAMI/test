# ✅ DailyDone - Todo App

> Simple. Fast. Productive.

---

## 📌 Overview

DailyDone is a lightweight Todo App built using **HTML, CSS, and JavaScript**.  
It allows users to manage daily tasks efficiently with a clean and user-friendly interface.

---

## ✨ Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 💾 Data stored in Local Storage
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
